## English

**Version:** 1.0.5
**Platform:** Web / Desktop

### Overview
A simple and effective application for managing your daily tasks. It helps you organize your work, set priorities, and track your achievements in an easy and flexible way.

### Main Features

- Create tasks easily
- Categorize tasks by project
- Set priorities
- Smart reminders
- Track achievements
- Simple and fast interface
