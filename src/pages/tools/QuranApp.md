<a name="english"></a>
## English

**Version:** 3.2.0
**Platform:** Android / iOS

### Overview
A comprehensive free application for reading the Holy Quran with an easy-to-use interface. It contains the full Quran with the ability to listen to recitations by various readers, in addition to search and bookmark features.

### Main Features

- Full Holy Quran
- Recitations by multiple readers
- Search in the Holy Quran
- Add bookmarks
- Night reading mode
- Works offline

### Download
https://play.google.com/store/apps/details?id=com.khaled.quran

## العربية (Arabic)

**الإصدار:** 3.2.0
**المنصة:** أندرويد / iOS

### نظرة عامة
تطبيق مجاني شامل لقراءة القرآن الكريم بواجهة سهلة الاستخدام. يحتوي على المصحف كاملاً مع إمكانية الاستماع للتلاوات بأصوات مختلف القراء، بالإضافة لميزات البحث والعلامات المرجعية.

### المميزات الرئيسية

- المصحف الشريف كاملاً
- تلاوات بأصوات قراء متعددين
- البحث في القرآن الكريم
- إضافة علامات مرجعية
- وضع القراءة الليلي
- يعمل بدون إنترنت
