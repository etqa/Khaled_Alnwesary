## English

**Version:** 1.0
**Platform:** Android / iOS

### Title
Quran App

### Short Description
Free mobile application for reading and reciting the Holy Quran

### Long Description
A free application for reading and reciting the Holy Quran with an easy-to-use interface and multiple features.

### Overview
A comprehensive Islamic application that provides a seamless experience for reading and listening to the Holy Quran. It features high-quality recitations from various famous reciters, prayer times, and Qibla direction.

### Video Tutorials
- [Khaled Alnwesary Channel][yt_chan]

### Main Features

- High-quality Quranic text with multiple fonts
- Audio recitations by world-renowned reciters
- Offline mode for reading and listening
- Advanced search and bookmarking system
- Multi-language translations and Tafsir
- Prayer times and accurate Qibla direction
- Clean and modern user interface
- No intrusive advertisements

### Title
تطبيق القرآن الكريم

### Short Description
تطبيق مجاني للهاتف لقراءة وتلاوة القرآن الكريم

### Long Description
تطبيق مجاني لقراءة وتلاوة القرآن الكريم بواجهة سهلة الاستخدام ومميزات متعددة.
